# Typography

Our typography system is built with readability and accessibility in mind, leveraging Tailwind CSS and the Typography plugin to create consistent, scalable text styles.

## Font Family

We use a system-first approach with Inter as our primary font:

<div className=" bg-gray-200 p-4 rounded-lg sb-unstyled">
  <p className="font-sans">The quick brown fox jumps over the lazy dog (Inter/System)</p>
  <p className="font-mono">const example = "Monospace font";</p>
</div>

```jsx
font-sans: Inter, system-ui, -apple-system, sans-serif
font-mono: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas
```

## Rich Text Content

When working with rich text content (like blog posts or articles), use the Typography plugin's `prose` classes:

<div className=" bg-gray-200 p-4 rounded-lg sb-unstyled">

<article className="prose prose-slate max-w-none">
  <h1>Article Title</h1>
  <p>Your content here with beautiful typography styling applied automatically. Notice how the spacing, line height, and other typographic details are handled by the Typography plugin.</p>
  <ul>
    <li>List item one</li>
    <li>List item two</li>
  </ul>
  <blockquote>
    Important callout message
  </blockquote>
</article>

</div>
```jsx
<article className="prose prose-slate max-w-none">
  <h1>Article Title</h1>
  <p>Your content here...</p>
  <ul>
    <li>List item one</li>
    <li>List item two</li>
  </ul>
  <blockquote>
    Important callout message
  </blockquote>
</article>
```
